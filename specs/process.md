## 大概流程

1. 新建小说 -> 进入小说创建流程/跳过流程自定义设定
2. 小说创建流程（标题确定->核心创意确定->题材和类型确定->世界观确定->人物确定->主线和主题确定->大纲确定：设计冲突和节奏） -> 创建卷
3. 跳过流程自定义设定 （初始化默认 namespace）-> 设置小说生成相关信息
4. 小说生成相关信息 (手动设置 标题、核心创意、题材和类型、世界观、人物、主线和主题、大纲等信息) -> 创建卷
5. 创建卷 -> 设置当前卷的大纲
6. 设置当前卷的大纲 -> 生成章节明细 -> 生成文章

## 注意点

如果小说类型是同人，可以选择原著
